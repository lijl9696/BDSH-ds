# 团购运营报表工具

本项目是一个本地桌面程序，用于处理美团、抖音导出的 Excel/CSV 报表：

- 可只导入美团、只导入抖音，或两个平台一起导入。
- 按配置表把平台字段映射成统一字段。
- 按门店名匹配门店负责人、大区和大区负责人。
- 按平台输出 Excel 报表：美团、抖音可使用各自的输出模板，互不干扰。
- 按排行榜配置输出手机竖版和汇报横版 PNG，也可按 `综合简报` 配置输出总览型图形化报告。
- 将标准化明细归档到本地 SQLite，便于后续做累计、同比、环比。

## 快速开始

1. 生成配置模板：

```bash
python3 tools/create_templates.py
```

2. 启动桌面程序：

```bash
python3 run_app.py
```

3. 在程序中选择：

- 配置表固定读取：`配置表/报表工具配置模板.xlsx`，界面不再要求每次选择。
- 美团输出模板固定读取：`配置表/美团输出报表模板.xlsx`，程序会按模板里的 sheet 结构写入结果；后续可增加 `配置表/抖音输出报表模板.xlsx`。
- 美团报表、抖音报表：按需选择一个或两个。

4. 点击 `生成 Excel 报表`，然后在榜单区域勾选需要的图片，点击 `生成报告图`。

## 配置表说明

配置表包含这些主要工作表：

- `字段映射`：配置每个平台原始字段到标准字段的映射、字段类型、是否输出、是否汇总、汇总方式、是否日期字段。
- `区域负责人`：按省份、城市配置城市负责人和归属大区。
- `门店对照`：按门店名补充门店 ID、省份、城市；如果平台报表本身已有省份/城市，可只维护门店名和 ID。
- `排行榜配置`：配置榜单名称、数据范围、统计维度、指标字段、排序、TopN、输出尺寸，以及可选的筛选字段/筛选值。
- `综合简报`：配置一张总览型报告图，包含门店数、评分、牌级分布、评价率、好评率、大区表现和重点门店列表。

负责人和大区匹配链路：

1. 平台报表只用标准字段 `门店名` 作为门店归属的唯一匹配字段。
2. 程序用 `门店名` 匹配 `门店对照`，取得 `门店负责人`。
3. 程序用 `门店负责人` 匹配 `区域负责人` 中的 `运营经理`，取得省份、城市和大区。
4. 匹配成功后，省份、城市、大区都以负责人所在区域为准；平台报表里的省份/城市只作为原始字段保留，不驱动归属。

## 原始报表读取

- Excel 原始报表会扫描前两行寻找配置表中的 `原始字段`，支持美团这类第一、第二行混合表头或合并单元格表头。
- `字段类型=数值` 的字段会自动清洗文本数字，例如逗号、货币符号、空格、`元` 后再转为数字。
- `门店ID` 等配置为文本的字段不会被自动转成数值。

### 汇总方式

字段映射支持 3 种汇总方式：

- `求和`：金额、订单数、券数、人数等累计指标。
- `平均`：评分、星级、转化率等均值指标。
- `计数`：记录数、门店数等需要统计条数的指标。

推荐在 `字段映射` 中新增 `汇总方式` 列填写以上值。为了兼容旧模板，也可以直接在 `是否汇总` 列填写 `求和`、`平均`、`计数`；旧的 `是` 会继续按 `求和` 处理。

文本类字段也可以做分类计数。例如 `经营评分牌级` 映射为标准字段 `牌级` 或旧字段 `牌级别`，程序会兼容识别；`金牌`、`银牌`、`铜牌` 单独分类，其他值统一合并为 `无等级`。

输出模板中，第 3 行是输出表头，第 4 行是规则。规则里出现 `分组`、`分类`、`维度` 或 `纬度`，该字段就会作为分组字段；指标字段可写成 `字段名 算法 条件`，例如 `门店名 去重计数 牌级=金牌`、`新增好评 求和/新增评价 求和`。

如果要生成“金牌大区门店数”图片，可在 `排行榜配置` 中配置：

- `统计维度`：大区
- `指标字段`：牌级别
- `排序`：desc
- `筛选字段`：牌级别
- `筛选值`：金牌

### 综合简报

`综合简报` 工作表用于生成更现代化的图形化总览。推荐配置：

- `简报名称`：门店评级与评价综合简报
- `数据范围`：美团
- `输出尺寸`：汇报横版，也可填 `手机竖版` 或 `全部`
- `门店字段`：门店名
- `分组字段`：大区
- `评分字段`：经营评分
- `星级字段`：美团星级
- `对比星级字段`：点评星级
- `等级字段`：牌级别
- `评价数字段`：新增评价
- `好评数字段`：新增好评
- `差评数字段`：新增差评；如果数据里没有该字段，程序会用 `新增评价 - 新增好评` 自动估算
- `订单字段`：核销单量
- `TopN`：重点门店列表展示数量

## 打包说明

安装 PyInstaller 后可打包：

```bash
pyinstaller --onefile --windowed --name 团购报表工具 run_app.py
```

Windows 和 macOS 需要分别在对应系统上打包。
