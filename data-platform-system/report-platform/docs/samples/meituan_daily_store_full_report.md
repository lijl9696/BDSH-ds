# 美团日全数据门店维度样本结构

主参考样本：

```text
日全数据-20260516-20260522-279904853-1780537525584349.xlsx
```

文件名不作为识别依据，只记录样本来源。

## 结构结论

- 这是当前唯一美团主参考结构；前期误传的周报/城市维度样本不作为配置依据。
- 工作簿只有 1 个 sheet，名称为 `sheet0`。
- 第 1 行是分组表头，例如 `客流分析`、`交易分析`、`评价分析`。
- 第 2 行是实际字段名。
- 第 3 行开始是数据。
- 数据粒度是 `日期区间 + 点评门店ID + 门店名称`。
- 日期字段是区间格式，例如 `2026-05-16~2026-05-22`。
- 有省份、城市字段，可以写入门店主数据。
- 该样本未出现“码类型”拆分多行，适合作为美团主报表结构参考。

## 口径说明

- 平台直接给出的比例指标，例如 `曝光访问转化率`、`意向转化率`，默认按平台值入库，汇总方式暂按平均。
- 可由分子/分母重算的经营比例，后续应优先使用派生指标规则计算。
- 累计类指标，例如累计收藏人数、累计评价数，默认使用 `latest` 口径，不应简单求和。
