# 抖音周度门店报表样本结构

样本文件：

```text
dyweek.xlsx
```

文件名不作为识别依据，只记录样本来源。

## 结构结论

- 工作簿只有 1 个 sheet，名称为 `Sheet1`。
- 第 1 行就是实际字段名。
- 第 2 行开始是数据。
- 数据粒度是 `周 + 门店ID + 门店名称`。
- 周字段是区间格式，例如 `2026-05-25~2026-05-31`。
- 有所在区域、所在省份、所在城市字段，可以写入门店主数据或后续维度。

## 关键维度字段

- 周
- 门店ID
- 门店名称
- 所在区域
- 所在省份
- 所在城市
